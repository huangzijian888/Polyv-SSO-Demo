## 快速开始
1. [下载 Jar 包](https://github.com/huangzijian888/Polyv-SSO-Demo/releases)
2. 终端/命令行运行以下命令
```shell
java -jar sso-0.0.1.jar
```
3. 浏览器打开 localhost:8080/index.html
